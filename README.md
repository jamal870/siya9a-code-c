# iDigi Sys Théorie Code C

Theorie- und Lern-App für Kategorie C in Marokko.

## Zweck

Dieses Projekt gehört zur Marke **iDigi Sys** und ist für die theoretische Ausbildung der Kategorie C in Marokko gedacht.

Die App unterstützt Fahrschüler und Fahrschulen bei der Vorbereitung auf die Theorieprüfung für schwere Fahrzeuge und Transport.

## Funktionen

- Theoriefragen für Kategorie C
- Lernmodus
- Prüfungssimulation
- Fortschrittsanzeige
- Kategorien und Themen
- Unterstützung für Fahrschulen und Fahrlehrer
- Vorbereitung auf Transport- und Berufsausbildung

## Zielgruppe

- Fahrschüler Kategorie C in Marokko
- Fahrschulen in Marokko
- Fahrlehrer für Kategorie C
- Transport- und Berufsausbildungsbereiche
